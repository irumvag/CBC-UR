Prompt Library

Ready-to-use prompts for common tasks

📝 Academic Writing

Improve your essays and papers

"I'm writing an essay about [topic]. Please review my thesis statement and suggest improvements for clarity and argumentation: [paste thesis]"

💡 Brainstorming Ideas

Generate project concepts

"I want to build an AI project that helps [target audience] with [problem area]. Generate 5 specific project ideas with brief descriptions of how they would work."

💻 Code Review

Get feedback on your code

"Please review this [language] code for bugs, best practices, and potential improvements. Explain any issues you find: [paste code]"

📖 Explain Concepts

Understand difficult topics

"Explain [concept] to me like I'm a university student with basic knowledge of [related field]. Use examples relevant to Rwanda when possible."

📊 Data Analysis

Analyze information

"Analyze this data and identify: 1) Key trends and patterns, 2) Outliers or anomalies, 3) Actionable insights. Data: [paste data]"

💬 Interview Prep

Practice for job interviews

"I have an interview for a [job title] position. Generate 10 likely interview questions and suggest strong answers based on my background: [describe experience]"
