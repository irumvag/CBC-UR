# Claude Builder Club — University of Rwanda

Official website for the Claude Builder Club at University of Rwanda, built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **React 18** with TypeScript
- **Vite** as build tool
- **Tailwind CSS** for styling
- **React Router v6** for navigation
- **Lucide React** for icons

## Design System

This website follows Anthropic/Claude's brand identity:

### Colors
- **Terracotta** `#da7756` - Primary accent
- **Pampas** `#F4F3EE` - Background
- **Ink** `#1A1714` - Primary text
- **Stone** `#8A8578` - Secondary text
- **Sage** `#7C9A82` - Success accent
- **Teal** `#4A8B8C` - Info accent

### Typography
- **Newsreader** - Serif font for headings
- **DM Sans** - Sans font for body text

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
