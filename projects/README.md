# CBC-UR Project Showcase

This folder contains projects and demos built by Claude Builder Club members at the University of Rwanda.

## How to Submit Your Project

1. Create a new folder with your project name (e.g., `my-claude-project/`)
2. Include your project code and a `README.md` describing what it does
3. Open a pull request to this repository

## Project Structure

Each project folder should contain:

```
your-project-name/
  README.md        # Project description, setup instructions, demo link
  src/             # Source code
  ...
```

## Guidelines

- Projects should use Claude or the Anthropic API in some way
- Include clear setup instructions in your README
- Add a brief description and any demo links
