# Claude Builder Club - University of Rwanda

Official website for the Claude Builder Club at University of Rwanda, a student organization promoting AI literacy through hands-on learning with Claude AI.

## About

Claude Builder Club is a student-led community that empowers University of Rwanda students to:
- Learn AI fundamentals through practical workshops
- Build real-world projects using Claude AI
- Connect with fellow students passionate about technology
- Develop skills for the AI-powered future

## Website Structure

```
CBC-UR/
├── index.html          # Home page
├── about.html          # About the club, mission, team
├── events.html         # Workshops, hackathons, meetups
├── projects.html       # Student project showcase
├── resources.html      # Learning materials and tutorials
├── join.html           # Membership registration
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # Interactive functionality
└── images/             # Images and assets
```

## Features

- **Responsive Design**: Mobile-first approach, works on all devices
- **Fast Loading**: Optimized for limited bandwidth connections
- **Accessible**: WCAG compliant, screen reader friendly
- **No Dependencies**: Pure HTML, CSS, and JavaScript
- **Easy to Maintain**: Simple file structure, no build process needed

## Quick Start

1. Clone the repository
2. Open `index.html` in a browser
3. That's it! No build step or dependencies required.

## Deployment

This site can be deployed to any static hosting service:

- **GitHub Pages**: Push to `gh-pages` branch
- **Netlify**: Drag and drop the folder
- **Vercel**: Import from GitHub
- **Any web server**: Upload files via FTP

## Customization

### Changing Colors
Edit the CSS custom properties in `css/styles.css`:
```css
:root {
  --primary: #6366f1;        /* Main brand color */
  --secondary: #8b5cf6;      /* Accent color */
  /* ... */
}
```

### Adding Events
Edit `events.html` and add new event cards following the existing pattern.

### Adding Projects
Edit `projects.html` and add new project cards with the `data-category` attribute.

### Updating Contact Info
Search for `250780000000` and `claude.builders@ur.ac.rw` to update WhatsApp and email.
